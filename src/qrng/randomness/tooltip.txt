Toeplitz Mixer: A deterministic randomness extractor based on the leftover hash lemma. It constructs a Toeplitz matrix using a seed derived from the source input's initial entropy (first 48 bytes). The input bitstring is then processed via matrix-vector multiplication over GF(2) to produce a condensed, uniform output distribution.

Polynomial Mixer: A universal hashing extractor that implements polynomial evaluation over a finite field. Input blocks are treated as coefficients of a polynomial, which is evaluated at a point determined by a seed derived from the source input (first 48 bytes). This method ensures pairwise independence in the output, effectively smoothing implementation-specific bias.

Von Neumann Extractor: A classical unseeded randomness extractor designed to eliminate fixed bias from a sequence of independent bits. It processes the input in non-overlapping pairs, mapping '01' to '1' and '10' to '0' while discarding '00' and '11'. This transformation converts a biased Bernoulli source into a uniform output stream, albeit with variable output length.

XOR Mixer: A computational extractor that whitens the input data by XORing it with a cryptographically secure pseudorandom keystream. The keystream is generated by an HMAC-DRBG seeded with the initial entropy of the source input (first 48 bytes), thereby masking statistical artifacts in the source data.

None: A pass-through mode that returns the raw input bitstring without any extraction or mixing, truncated to the nearest byte alignment.
